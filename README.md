# godev

Desafio de Programação GoDev

Para rodar a aplicação é necessário ter o JavaFX instalado com a sua biblioteca.

Baixar o openJDK 11.0.2 e extrair na pasta C:\Program Files\Java

Adicionar as variavel de ambiente de sistema o JAVA_HOME com o caminho C:\Program Files\Java\jdk-11.0.2

Adicionar a variavel de ambiente de sistema Path o caminho %JAVA_HOME%\bin

Baixar o openFX 11.0.2 e extrair para a pasta C:\java-libs

Adicionar a variavel de ambiente de sistema o PATH_TO_FX com o caminho C:\java-libs\javafx-sdk-11.0.2

Baixar o arquivo godev.rar pelo link: 
https://www.dropbox.com/s/91zwvmwrrqhtyix/godev.rar?dl=0

Para rodar a aplicação é só executar o arquivo GoDev.bat

-------------------------------------------------------------------------------------------------

Na tela principal do programa você escolhe a aba de cadastro e cadastra quantos alunos quer e seus respectivos nomes.

Cadastra as 2 salas e suas respectivas capacidades.

Cadastra as 2 cafeterias e suas respectivas capacidades.

Na aba de consulta você pode consultar o nome do aluno para descobrir as salas em cada um dos perodos do curso e o local do café.

Pode consulta os as salas para ver quais alunos estarão em cada período.´
